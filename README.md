# Cashmere

Cashmere is a high performance in-memory caching solution.

## Installation

Add Ccashmere as a dependency to your `mix.exs` file:

```elixir
def deps do
  [
    {:cashmere, "~> 0.1"}
  ]
end
```

Then, run `mix deps.get` in your shell to fetch the dependencies.
